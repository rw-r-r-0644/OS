# OS  

### How to build  
#### Requirements:  
- GNU make  
- i386-elf gcc toolchain (you can find one here: http://wiki.osdev.org/GCC_Cross-Compiler#Prebuilt_Toolchains)  
- nasm  
- grub and grub-mkrescue  
#### Build instructions:  
1. Download the main repo  
2. Run "make" in the root folder
